# E-commerce SyntaxWear

Este é um projeto de e-commerce front-end, desenvolvido para a loja SyntaxWear. Ele apresenta uma estrutura básica para um site de vendas online, com seções para produtos, banners, ícones e estilos bem definidos, com foco em tênis e sneakers.

## Funcionalidades

*   **Cabeçalho Responsivo:** Navegação principal com categorias (Masculino, Feminino, Outlet) e links secundários (Nossas Lojas, Sobre, Minha Conta, Ajuda, Carrinho).
*   **Seção Hero:** Destaque visual com título, subtítulo e chamadas para ação.
*   **Categorias de Produtos:** Exibição de categorias de produtos como Casual, Esporte, Moderno e Futurista.
*   **Grade de Produtos:** Layout para exibição de diversos produtos.
*   **Rodapé Completo:** Inclui formulário de inscrição para newsletter, links para redes sociais (Instagram, WhatsApp, TikTok, Facebook) e navegação detalhada.

## Tecnologias Utilizadas

-   **HTML:** Estrutura fundamental das páginas web.
-   **CSS:** Estilização e design visual.

## Como Executar Localmente

Para visualizar o projeto em seu navegador, basta abrir o arquivo `index.html` diretamente em seu navegador web preferido.
